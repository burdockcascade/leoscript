Leo Script
==========

## What is Leo Script?

Leo Script is a simple scripting language that is designed to be easy to use and easy to learn.

## Features
* Simple syntax
* Classes & Modules
* No semicolons
* No curly braces

## Design Goals
* Easy to use
* Easy to learn
* Easy to read
